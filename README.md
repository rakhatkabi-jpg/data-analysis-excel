# Блок 2: Анализ данных и моделирование в Excel

Репозиторий содержит финальные практические задания, расчеты и аналитические модели, выполненные в Microsoft Excel.

## 🛠️ Что было сделано:
- Проведена очистка, форматирование и подготовка сырых данных к анализу.
- Настроены сложные расчеты с использованием функций (`ВПР/VLOOKUP`, `ИНДЕКС/ПОИСКПОЗ`, `ЕСЛИ`, `СУММЕСЛИМН`).
- Построены сводные таблицы (Pivot Tables) для быстрой агрегации больших массивов данных.
- Создан интерактивный дашборд с использованием срезов и динамических диаграмм.

## 📁 Структура файлов в репозитории:
- `data_analysis_model.xlsx` — итоговый файл с расчетами, таблицами и визуализацией.
- `README.md` — описание проекта и ключевых этапов работы.
